
# tap

  tap on elements. bypasses the click delay on mobile.

## Installation

    $ component install component/tap

## API

### Tap(el, fn)

Initialize a new tap event on `el` that triggers `fn`.

```js
tap(el, fn);
```

### .unbind()

Unbind the tap event.

```js
tap.unbind();
```
